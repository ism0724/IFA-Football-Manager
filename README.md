# IFA - Football School Manager

Application de gestion administrative pour une école de football.

## Fonctionnalités
- [ ] Authentification sécurisée Admin(Staff/Coach)
- [ ] Gestion des joueurs (Inscriptions, Profils)
- [ ] Gestion des catégories (U11, U13, etc.)
- [ ] Tableau de bord statistique

## Technologies
- Langage : PHP 8
- Base de données : MySQL
- Frontend : HTML5 / CSS3 / Bootstrap
